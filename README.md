# Machine Learning

关于机器学习的建议

如果你是数学或者统计专业的学生

先把数学分析，高等代数，概率论，数理统计，多元统计分析修完

大概完成时间为大二

在这个过程中，学好C++ 和python

这估计已经阻碍了很大一部分人进入该领域

不过你还可以走一些捷径

比如 MIT 大佬写的 DeepLearning 的书

比如 UCB 大佬的公开课

来迅速 get 到这部分 knowledge

然后看周志华大大的西瓜书

接着看一些应用

比如 Stanford 的课程

来检测你的学习
